## Getting Started

### First, install dependencies:

```bash
npm ci
# or
yarn --frozen-lock
```

### Second, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Last, run the test:

```bash
npm run cypress:open
# or
yarn cypress:open
```
